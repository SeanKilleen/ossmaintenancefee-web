The following are a couple of of end-user license agreements (EULA) that you could use in your project to require the Maintenance Fee.

<Aside type='note' title='IANAL'>
These EULAs are currently in draft. We have our lawyer reviewing them and will update them (and this notice) soon.
</Aside>



## Require the Maintenance Fee EULA

This license requires all revenue generating consumers to pay the Maintenance Fee. Update the `[Project Name]` and `[Software Name]` values that are appropriate to describe your software and project name. Often the two will be the same.

```
End User License Agreement (EULA)

This End User License Agreement ("Agreement") is a legal agreement
between you ("User") and [Project Name] ("Project") for the use of
[Software Name] ("Software"). By installing, accessing, or using the
Software, you agree to be bound by the terms of this Agreement.

1. License Grant

Project grants User a non-exclusive, non-transferable license to use
the Software, subject to the terms and conditions of this Agreement.

2. Open Source Maintenance Fee

User acknowledges that the Software is open-source and requires ongoing
maintenance and support. To ensure continued development and stability,
User agrees to pay the Open Source Maintenance Fee ("Fee") as determined
by the Project, if the User generates revenue using the Software. The
Fee must be paid as determined by the Project to maintain the right to
use the Software.

3. Termination

Failure to pay the Fee will result in the termination of this license.
Upon termination, User must cease all use of the Software and remove it
from all systems.

4. Disclaimer of Warranty

The Software is provided "as is" without warranties of any kind. Project
disclaims all warranties, express or implied, including but not limited
to warranties of merchantability and fitness for a particular purpose.

5. Limitation of Liability

Project shall not be liable for any indirect, incidental, or consequential
damages arising out of the use of the Software.

By using the Software, User acknowledges and agrees to the terms of this
Agreement, including the payment of the Open Source Maintenance Fee.
```


## Require the Maintenance Fee except for Customers EULA

This license requires all consumers to pay the Maintenance Fee unless they are customers of the company developing the project. Update the `[Company Name]`, `[Project Name]`, and `[Software Name]` values that are appropriate to describe your company, project, and software. Often the `[Project Name]` and `[Software Name]` will be the same.

```
End User License Agreement (EULA)

This End User License Agreement ("Agreement") is a legal agreement 
between you ("User") and [Project Name] ("Project") produced by
[Company Name]("Company") for the use of [Software Name] ("Software").
By installing, accessing, or using the Software, you agree to be bound
by the terms of this Agreement.

1. License Grant

Project grants User a non-exclusive, non-transferable license to use the
Software, subject to the terms and conditions of this Agreement.

2. Open Source Maintenance Fee

User acknowledges that the Software is open-source and requires ongoing
maintenance and support. To ensure continued development and stability,
User agrees to pay the Open Source Maintenance Fee ("Fee") as determined
by the Project, if the User generates revenue using the Software. However,
if the User is a customer of the Company that produces the Project, the
User is not required to pay the Fee.

3. Termination

Failure to pay the Fee will result in the termination of this license.
Upon termination, User must cease all use of the Software and remove it
from all systems.

4. Disclaimer of Warranty

The Software is provided "as is" without warranties of any kind. Project
disclaims all warranties, express or implied, including but not limited
to warranties of merchantability and fitness for a particular purpose.

5. Limitation of Liability

Project shall not be liable for any indirect, incidental, or consequential
damages arising out of the use of the Software.

By using the Software, User acknowledges and agrees to the terms of this
Agreement, including the payment of the Open Source Maintenance Fee, unless
exempt as a customer of the Company that produces the Project.
```
